# 摸金模拟器 (Gold Simulator)

回合制文字搜打撤策略游戏。把逃离塔科夫的核心循环（搜刮→战斗→撤离→治疗）做成浏览器里 5 分钟一局的回合制体验。

## 🎮 怎么玩

**直接打开（推荐）：** https://whc327829624-dotcom.github.io/Gold-simulation/tarkov-lite.html

**或者本地：** 下载 `tarkov-lite.html`，双击用浏览器打开。

## 🕹️ 核心玩法

进图 → 走格子搜物资 → 碰到敌人打架 → 找撤离点回家 → 治伤 → 买装备 → 再来一局

死亡装备全丢，保险箱里的东西能保住。伤势不会自动好，得手动用药治。
